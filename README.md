# 101-series

## Environment Setup
```
# create a separate conda env and activate it
conda create -n pyspark-101 python=3.8
conda activate pyspark-101

# install required libraries 
pip install -r requirements

# OPTIONAL: create a jupyter kernel
python -m ipykernel install --user --name pyspark-101  --display-name "pyspark-101"
```
